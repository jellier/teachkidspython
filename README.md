# 2020寒假python编程教学计划

## 教学目标：
> 面向0基础的孩子，初步认识真正的编程语言

> 分析问题 --> 解决问题 --> 讨论及分享

> 独立实现一段可执行代码

## 教学任务

> 入门：认识编程的基本概念 -- 变量，条件，循环，列表，函数等

> 进阶：用编程思维解决数学问题

> 提高：编写简单的爬虫程序
